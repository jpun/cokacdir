# PR #9 메인테이너 의견

PR 제출해 주셔서 감사합니다.

봇 UI 메시지의 다국어 지원 아이디어는 매우 좋습니다. 채팅별 언어 설정이 가능한 구조도 잘 설계되어 있다고 생각합니다. i18n 모듈 구조, Lang enum, `/lang` 커맨드 등 전반적인 설계가 깔끔합니다.

다만, PR의 base 코드와 현재 main 사이에 구조적 차이가 크고(약 1500줄 차이), 충돌 지점이 다수 존재하여 직접 머지가 어려운 상황입니다. 또한 현재 main에만 존재하는 기능들(스케줄, 모델 선택, 디버그 등)의 문자열도 i18n 처리가 필요합니다.

이 PR의 아이디어와 설계를 기반으로, 현재 main 코드에 맞춰 별도로 구현하는 것이 적절하다고 생각합니다.

좋은 기여 감사드립니다.
