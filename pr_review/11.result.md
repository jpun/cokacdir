# PR #11 메인테이너 의견

PR 제출해 주셔서 감사합니다.

다양한 AI 백엔드를 플러그인처럼 지원할 수 있는 추상화 레이어라는 설계 방향은 좋다고 생각합니다. CancelToken의 메서드 캡슐화나 시스템 프롬프트를 const로 추출한 점 등 개선 요소도 있습니다.

## 현재 상태에 대한 의견

추상화 레이어를 정의해 두었지만 실제로 이를 사용하는 코드 경로가 없어, 현 시점에서는 동작하지 않는 코드입니다. 실제 백엔드 전환이 동작하려면 후속 작업 범위가 상당히 큽니다. 추상화가 실제로 동작하는 단계까지 완성되었을 때 다시 검토하는 것이 적절하다고 생각합니다. 그때 이 PR의 설계가 좋은 참고가 될 것입니다.
